# Desafio Técnico – Front-End (Next.js)

Este é um projeto desenvolvido em Next.js com um CRUD completo de produtos. Ele foi criado como um teste técnico de desenvolvedor front-end para a empresa MaxUp Consultoria.

---

## Como rodar o projeto

1. Instalar as dependencias

`npm install`

2. Rodar o localmente no modo de desenvolvimento

`npm run dev`

3. Abrir no navegador

`localhost:3000`

---

## Tecnologias utilizadas

- Next.js
- Tailwind
- shadcn-ui
- Lucide Icons

## Observações

- O projeto utiliza a [FakeStore API](https://fakestoreapi.com/)
